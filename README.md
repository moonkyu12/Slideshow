# Classroom Slideshow Extension

Google Classroom과 Google Drive에 첨부된 PDF, PPT 프레젠테이션 파일을 다운로드 과정이나 별도의 뷰어 프로그램 없이 브라우저 내에서 즉시 **프리미엄 전체화면 슬라이드쇼**로 감상할 수 있게 해주는 Chrome 확장프로그램이다. 에헤 잘만들었제

## 주요 기능

- **자동 버튼 주입 (UI Injection):** 구글 클래스룸의 과제 첨부파일 미리보기 창에 들어가면 자동으로 화면 우측 하단에 스텔스(Stealth) 방식으로 [이 파일로 슬라이드쇼 시작] 버튼이 생성됩니다.
- **자체 렌더링 엔진 (PDF.js 내장):** 확장프로그램 내부에 자체 엔진을 내장하여 그 어떤 외부 보안 제약(CSP)이나 네트워크 차단 없이 매우 빠르고 안전하게 파일을 읽어냅니다.
- **백그라운드 프록시 (CORS 우회):** 구글 드라이브(문서) 링크의 강력한 CORS 및 파일 다운로드 리다이렉트 정책(`usercontent.google.com`)을 백그라운드 서비스 워커를 통해 무력화시켜 모든 비공개 파일에도 원활하게 접근합니다.
- **프리미엄 다크 테마 UI:**
  - 시선을 분산시키지 않는 세련된 글래스모피즘(Glassmorphism) 테마
  - 파일 로딩 진행률 트래커 및 부드러운 슬라이드 전환 애니메이션
  - 마우스를 올릴 때만 나타나는 직관적인 슬라이드쇼 전용 네비게이션
  - 모바일 터치(스와이프) 지원 및 키보드 단축키 지원

## 설치 방법 (개발자 모드)

현재 이 확장프로그램은 스토어 등록 전 "개발자 버전" 상태입니다.

1. 크롬 브라우저를 열고 주소창에 `chrome://extensions/` 를 입력해 접속합니다.
2. 화면 우측 상단의 **"개발자 모드"** 토글 스위치를 켭니다.
3. 좌측 상단에 나타난 **"압축해제된 확장 프로그램을 로드합니다(Load unpacked)"** 버튼을 클릭합니다.
4. 이 소스코드가 들어있는 `Slideshow` 폴더를 선택합니다.
5. 설치가 완료되면 확장프로그램 목록에 "Classroom Slideshow" 카드가 추가됩니다!

## 사용 방법

1. 구글 클래스룸 접속 후 강의나 과제에 첨부된 PDF, PPTX 파일을 클릭해 미리보기 화면을 엽니다.
2. 화면 우측 하단에 파란색 **[이 파일로 슬라이드쇼 시작]** 버튼이 생성된 것을 확인하고 클릭합니다.
3. 멋진 애니메이션과 함께 새 탭에 전체화면 프레젠테이션 뷰어가 열립니다.

### 단축키 안내

슬라이드쇼 창에서는 다음과 같은 단축키가 작동합니다:
- **다음 슬라이드:** `→`, `↓`, `Space`, `Enter`
- **이전 슬라이드:** `←`, `↑`, `Backspace`
- **전체화면 전환:** `F`
- **슬라이드 목록 전체 보기(그리드 모드):** `G`
- **슬라이드쇼 종료:** `ESC`
- *(마우스 휠 스크롤 및 터치 화면 좌/우 스와이프도 완벽하게 지원합니다)*

## 🛠 아키텍처 및 구성 요소

- `manifest.json`: 확장프로그램 설정 및 권한, 호스트 우회(`host_permissions`) 룰 정의
- `content.js` / `content.css`: 구글 클래스룸 DOM 변경사항을 감시(`MutationObserver`)하고 슬라이드쇼 버튼을 주입하는 스크립트
- `background.js`: 사용자 쿠키와 백그라운드 권한을 활용해 CORS 제약을 뚫고 파일(Blob)을 다운로드해 뷰어에 Base64 형태로 전달해주는 프록시 워커
- `viewer/`: 슬라이드쇼를 렌더링하는 자체 독립 웹 애플리케이션 화면 (`viewer.html`, `viewer.js`, `viewer.css`)
- `viewer/lib/`: Mozilla의 정통 PDF 웹 렌더링 라이브러리인 PDF.js 로컬 구동 코어 파일들
